<h1 align="center">🚀 Meu Primeiro Projeto HTML e CSS</h1>

<p align="center">
  Meu primeiro projeto desenvolvido utilizando apenas HTML e CSS básico.
</p>

<hr>

<h2>📖 Sobre o Projeto</h2>

<p>
Este projeto foi criado com o objetivo de praticar os conceitos fundamentais de desenvolvimento web,
utilizando apenas HTML para a estrutura da página e CSS para a estilização.
</p>

<p>
Durante o desenvolvimento, pude aprender sobre:
</p>

<ul>
  <li>Estrutura básica de um documento HTML</li>
  <li>Tags semânticas</li>
  <li>Estilização com CSS</li>
  <li>Cores, fontes e espaçamentos</li>
  <li>Organização de conteúdo em páginas web</li>
</ul>

<h2>🛠️ Tecnologias Utilizadas</h2>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
</ul>

<h2>🎯 Objetivo</h2>

<p>
Este projeto faz parte da minha jornada de aprendizado em desenvolvimento web e representa meus primeiros passos na criação de páginas utilizando tecnologias básicas da web.
</p>

<h2>📂 Como Executar</h2>

<ol>
  <li>Clone este repositório</li>
  <li>Abra o arquivo <code>index.html</code> no navegador</li>
</ol>

<h2>📚 Aprendizados</h2>

<p>
Com este projeto, desenvolvi uma melhor compreensão da estrutura de páginas web e dos conceitos iniciais de estilização utilizando CSS.
